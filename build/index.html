Projects

Knight's Travails

Knights Travails is an app intended to calculate and display the minimum number of moves and route a chess piece would need to get from a start point to an end point on a chess board. This app uses a custom tree/graph like data structure and breadth first search to calculate the knights moves.

Weather App

Weather app designed using WeatherAPI to provide detailed weather forecast information, including 12 hour and 3-day forecasts. The app includes a search bar with an autocomplete feature and the ability to toggle between Fahrenheit and Celsius.
